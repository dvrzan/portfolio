Clima

Weather app

Apart from showing current and hourly weather, Clima shows information about sunset and sunrise times, wind speed, minimum, maximum and real feel temperatures. Built with Swift using Dark Sky API. Features simple UI, custom condition backgrounds, and a list of saved locations.

The app is currently in development.

Analyzar

Twitter sentiment analyzer app

Analyzar provides insight on how people feel on a certain topic, person, or company. Built with Swift using Twitter API to provide sentiment. Features simple UI, fast analysis, and a simple introduction to machine learning series.

Want to learn more about the project? You can check it out HERE.

Skymaster

2D Space shooter game

Skymaster is a fast-paced 2D space shooter game where you test your abilities to become the next executive officer aboard the Skymaster spaceship. Built using Unity and C#. Features a single and local co-op playing styles, different power-ups to aid you in the battle, and an awesome exploding sounds and graphics.
